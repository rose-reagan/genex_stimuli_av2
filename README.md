# GenEx Stimuli Bank (Audio/Visual v2)

Stimuli repository for the GenEx experiment.

## Directory Structure

- **`images/`** - Image stimuli files
- **`mp3/`** - Audio stimuli files (MP3 format)
- **`mp4/`** - Video stimuli files (MP4 format)

## Usage

Upload your experiment stimuli files to the appropriate directories based on their type.

### Adding Files

1. Clone the repository
2. Add your stimuli files to the relevant directories
3. Commit and push your changes

```bash
git clone https://github.com/rose-reagan/genex_stimuli_av2.git
cd genex_stimuli_av2
# Add files to images/, mp3/, or mp4/ directories
git add .
git commit -m "Add stimuli files"
git push
```
